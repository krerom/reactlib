# reactlib
Just a fun React component library project to try things out and build reusable UI pieces. v1 — forks and contributions are more than welcome.

## Installation

For local development or testing, you can install the library using the packed `.tgz` file:

```bash
npm install path/to/reactlib-1.0.0.tgz
```
